Учебный журнал

Учебный проект (курсовая работа), представляющий собой систему ведения учебного журнала.  
Проект разработан в рамках освоения практических навыков программирования и работы с базой данных.

 Цель проекта
Закрепление теоретических и практических знаний по разработке программных приложений, включая:
- работу с логикой приложения
- хранение и обработку данных
- структурирование кода
- использование системы контроля версий Git

 Функциональные возможности
- добавление и хранение данных (студенты / предметы / оценки)
- просмотр и управление записями журнала
- базовая логика обработки данных
- (при необходимости можешь дописать конкретные функции)

 Используемые технологии
- Python
- SQLite3
- Git / GitHub
